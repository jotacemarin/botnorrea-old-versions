# botnorrea
a simple discord bot write in javascript

## dependencies
* `axios` ^0.19.0
* `bunyan`^1.8.12
* `discord.js` ^11.5.1
* `emoji-unicode` ^1.0.10
* `ffmpeg-binaries` ^4.0.0
* `node-opus` ^0.3.2
* `redis` ^2.8.0
* `ytdl-core` ^0.29.5

## scripts availables
* `npm run dev` run proyect with dev config
